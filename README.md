# What is this?
This extension toggles tasks in Markdown like Obsidian.

## How To Use
Place your cursor on a line and press the shortcut key to cycle through the following states:

```
Plain text → List (- ) → Unchecked task (- [ ] ) → Checked task (- [x] ) → Plain text → ...
```

Shortcut key:
- Mac: `Cmd+Shift+Enter`
- Windows/Linux: `Ctrl+Shift+Enter`

Multiple selections are supported — each line is toggled independently.
