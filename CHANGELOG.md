# Change Log

## [0.0.3] - 2026-05-06

### Fixed
- Checked task (`- [x] `) now correctly returns to plain text instead of a list item

### Changed
- Shortcut key changed to `Cmd+Shift+Enter` (Mac) / `Ctrl+Shift+Enter` (Windows/Linux) to avoid conflict with VSCode default keybindings

## [0.0.2]

### Changed
- Toggle cycle order changed to: plain text → list → unchecked task → checked task → plain text
